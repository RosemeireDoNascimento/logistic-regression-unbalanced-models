# Códigos da Dissertação

Este repositório contém os códigos utilizados na dissertação:

"Regressão Logística Aplicada a Dados Desbalanceados: Modelagem e Avaliação Preditiva e Inferencial"

Autor: Rosemeire do Nascimento Santos

## Conteúdo

- Simulação Monte Carlo
- Regressão logística
- Técnicas de balanceamento (SMOTE, NearMiss, Tomek Links, etc.)
- Avaliação por métricas de classificação e estimação

## Requisitos

- R >= 4.3
- pacotes: caret, pROC, smotefamily, glmnet...

## Reprodução

Execute os scripts na pasta scripts na seguinte ordem:

1. simulacao.R
2. modelos.R
3. metricas.R
